# Nautilux

## Back-end Django:
* Python version 2.7
* Django version 1.11

### Lancement du back:

Dans le dossier back, activez l'environnement virtuel:
source env/bin/activate

Dans le dossier nautilux_back, lancez le serveur Django:
python manage.py runserver
